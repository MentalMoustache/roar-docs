# `level_brackets.xml`

This file contains the xp required to obtain each level in the game. It typically looks like this.

~~~
<chameleon_data>
  <level_brackets>
    <level_bracket level="1"  xp="20" />
    <level_bracket level="2"  xp="30" />
    <level_bracket level="3"  xp="50" />
    ...
  </level_brackets>
</chameleon_data>
~~~
